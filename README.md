# Digital Business Card

Digital Business Card is a simple React application built with Vite that presents a personal profile as an online business card. It displays key information such as name, role, contact details, and social media links in a clean, responsive layout.

## Features

- Personal photo and basic profile information
- Short description/bio section
- Contact buttons (email, website, etc.)
- Social media icons linking to external profiles
- Simple, modern styling with CSS

## Technologies

- React
- Vite
- JavaScript (ES6+)
- CSS.
